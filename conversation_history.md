# История разработки: Платформа и CRM для экспертов нутрициологии (Herbalife)

## 1. Начальный запрос пользователя
**Цель**: Перевести ручной сбор заявок из соцсетей (ВК, Telegram) в автоматизированную систему.
**Ключевые модули**:
- Адаптивный лендинг (Apple Health style).
- Квиз-анкета (5-7 вопросов).
- Раздел событий (календарь).
- CRM: Карточка клиента, Личный кабинет марафона, Dashboard для эксперта.
**Стиль**: Белый, мягкий зеленый (#7BC143).

## 2. Предложение Antigravity (Implementation Plan v1)
- Предложен стек: Next.js + Tailwind + Supabase.
- Описаны кейсы Вадима Дорожкина (умные ссылки) и Людмилы Маликовой (коллажи До/После).
- Внедрена система Visual Feedback.

## 3. Обратная связь и правки пользователя
**Критическая правка по безопасности**:
- Переход с Supabase на **Appwrite на российских серверах** (соблюдение ФЗ-152).
- Все эксперты находятся в Сосновом Бору (отказ от сложной геолокации в пользу слагов).

**Правки по дизайну**:
- Строгая палитра Herbalife: `#7BC143` (основной), `#387C2B` (ховер), `#262324` (текст), `#F9F8F4` (подложка).

**Правки по CRM**:
- Интерактивная хроника (Timeline).
- Продуктовый таймер (21 день на банку коктейля).
- Режим «Инкогнито» (блюр лиц на фото).
- Геймификация: «Огоньки» (Streaks) как в Duolingo, индикаторы прогресса.

**Правило взаимодействия**:
- Приступать к следующему разделу плана можно **СТРОГО** после разрешения пользователя. (Зафиксировано в `AGENTS.md`).

## 4. Выполненные действия (Раздел 1)
1. **Инициализация**: Развернут проект Next.js 14+/16.
2. **UI-Система**: Настроены темы Tailwind v4 и базовые компоненты (Button, Card) в стиле Apple Health.
3. **Роутинг**: Реализована Multi-tenant структура `/[slug]` для разделения пространств экспертов.
4. **Хаб**: Сверстана профессиональная визитка эксперта с блоками доверия.
5. **ER-диаграмма**: Спроектирована схема данных (Clients, Logs, Inventory, Timeline).

## 5. Текущий статус
- **Раздел 1 (Инфраструктура и Фундамент) — ВЫПОЛНЕНО (100%)**. Установлен SDK Appwrite, создана конфигурация и `.env.example`.
- Раздел 2 (Публичный Хаб и Квиз) — визуал хаба готов, Квиз и **механизм персонализации через URL (Smart Links)** полностью реализованы.
